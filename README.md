# Portfolio - Christian Dongueu Feyem

Un portfolio moderne développé avec React et NestJS.

## Technologies Utilisées

### Frontend
- React 18
- TypeScript
- CSS Moderne avec variables et animations
- Axios pour les appels API

### Backend
- NestJS
- TypeScript
- API RESTful

## Installation

### Prérequis
- Node.js (version 14 ou supérieure)
- npm ou yarn

### Backend

```bash
cd portfolio-fullstack/backend
npm install
npm run start:dev
```

### Frontend

```bash
cd portfolio-fullstack/frontend
npm install
npm start
```
